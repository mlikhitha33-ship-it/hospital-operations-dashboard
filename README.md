# 🏥 Hospital Operations Intelligence Dashboard

An interactive healthcare analytics dashboard built using Streamlit, Pandas, and Plotly to analyze hospital emergency room operations, patient admissions, wait times, and department referrals.

## About the Project

I built this project to explore how healthcare data can be transformed into meaningful insights through interactive dashboards.

The dashboard analyzes more than **9,200 patient records** and provides a clear view of key operational metrics such as patient admissions, wait times, satisfaction scores, and referral patterns across hospital departments.

The goal was to create a user-friendly dashboard that allows healthcare administrators and analysts to quickly identify trends and make data-driven decisions.

## Live Demo

🔗 Streamlit Dashboard:https://hospital-operations-dashboard-mgwgnschrh8npzihzhjg3p.streamlit.app/

## Key Features

* Interactive filters for department, age group, admission status, and date range
* KPI cards displaying important hospital metrics
* Monthly admissions trend analysis
* Department referral analysis
* Patient satisfaction monitoring
* Wait time analysis
* Interactive charts and visualizations using Plotly

## Business Questions Addressed

This dashboard helps answer questions such as:

* How many patients visited the hospital during a given period?
* What is the average patient wait time?
* Which departments receive the highest number of referrals?
* What percentage of patients are admitted?
* How satisfied are patients with the services provided?
* Are there any noticeable trends in monthly admissions?

## Running the Project Locally

Clone the repository:

```bash
git clone https://github.com/mlikhitha33-ship-it/hospital-operations-dashboard.git
```

Move into the project directory:

```bash
cd hospital-operations-dashboard
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
